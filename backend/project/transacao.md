## Criação de Transação

Para criar uma transação, é necessário ter uma **conta bancária cadastrada**.

1. Para cadastrar uma conta bancária, utilize a rota de **cadastro de contas**, onde devem ser informados:

    - Banco
    - Descrição
    - Conta
    - Agência
    - Chaves Pix cadastradas

2. Após cadastrar uma conta bancária, é necessário definir a **categoria da transação**, que determina se a operação será de **débito** ou **crédito**.

3. Depois de selecionar a categoria, informe as **formas de pagamento** para concluir a transação.

4. Essa transação também pode gerar **cobranças futuras**, como parcelamento no crédito.
